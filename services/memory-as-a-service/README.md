# Memory as a Service

This service provides semantic memory storage with vector embeddings for AI applications.

## Features
- Memory storage and retrieval
- Semantic search capabilities
- Vector embeddings
- Memory categorization and tagging
- Bulk operations
- Memory statistics and analytics

## Integration
This service connects to Supabase Edge Functions and provides MCP-compatible tools for memory operations.

## MCP Tools
- `create` - Create a new memory entry
- `get` - Retrieve a specific memory by ID
- `update` - Update an existing memory entry
- `delete` - Delete a memory entry
- `list` - List memories with pagination and filtering
- `search` - Semantic search across memories
- `stats` - Get memory statistics and analytics
- `bulk-delete` - Bulk delete multiple memories
- `health` - Check the health of the memory service
