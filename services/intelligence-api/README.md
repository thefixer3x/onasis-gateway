# Intelligence API

This service provides business intelligence and analytics capabilities.

## Features
- Tag suggestion for content
- Related content discovery
- Duplicate detection
- Insight extraction
- Pattern analysis
- Report generation
- Outcome prediction

## Integration
This service connects to Supabase Edge Functions and provides MCP-compatible tools for intelligence operations.

## MCP Tools
- `suggest-tags` - Suggest relevant tags for content
- `find-related` - Find related content or items
- `detect-duplicates` - Detect duplicate or similar content
- `extract-insights` - Extract insights from data or content
- `analyze-patterns` - Analyze patterns in data
- `generate-report` - Generate analytical reports
- `predict-outcomes` - Predict outcomes based on historical data
- `health-check` - Check the health of the intelligence service
